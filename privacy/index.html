プライバシーポリシー

サービス名: 処方メモ (Yakumemo) 事業者: 株式会社tmedilab (以下「当社」といいます) 初回公開日: 2026年6月15日 (下書き) 最終改定日: 2026年7月8日

第1条 (適用範囲)

本プライバシーポリシー (以下「本ポリシー」といいます) は、当社が提供する iOS アプリケーション「処方メモ」(以下「本アプリ」といいます) および当社が運営するウェブ版「処方メモ」その他本ポリシーへのリンクを表示するウェブページ (以下、本アプリと総称して「本サービス」といいます) における、ユーザーの個人情報の取り扱いについて定めます。本アプリとウェブ版は同一のアカウントおよびデータ基盤を共有します。

本アプリをダウンロード・インストールし、または本サービスを利用された場合、ユーザーは本ポリシーに同意したものとみなします。

第2条 (収集する個人情報)

当社は、本サービスの提供にあたり、以下の個人情報を取得することがあります。

(1) ユーザーが直接提供する情報

  • メールアドレス — アカウントの登録およびログイン認証のために収集します。
  • ニックネーム — コミュニティ機能における表示名として収集します。本名の入力は求めません。
  • パスワード — ハッシュ化されて Supabase 認証基盤に保管され、原文は当社では一切閲覧できません。
  • コミュニティ投稿コンテンツ — ユーザーがコミュニティ機能を通じて投稿するレビュー本文、評価値 (4軸)、タグ、口コミ等。これらは他のユーザーから閲覧可能な「公開コンテンツ」となります。
  • 服薬履歴データ — ユーザーが本アプリ上で記録する、自身および家族メンバーの服薬履歴 (お薬名、服用日時、服用量、メモ等)。これらは原則として公開されず、ユーザー本人およびユーザーが登録した家族メンバーの管理目的でのみ利用されます。
  • 家族メンバー情報 — ユーザーが登録する家族メンバーの表示名等。表示名は本人名以外の任意の文字列を推奨します。

(2) 本アプリの動作に伴い自動的に取得する情報

  • ユーザーID — Supabase が発行する内部識別子。当社サービス内でのみアカウント識別に使用します。
  • ファイルタイムスタンプ — Apple Required Reason API C617.1 に該当。SwiftData の永続化ストアおよびファイル管理 API がアプリ内部で利用します。
  • システム起動時刻 — Apple Required Reason API 35F9.1 に該当。MetricKit による障害計測およびキャッシュ鮮度判定に利用します。
  • UserDefaults — Apple Required Reason API CA92.1 に該当。アプリの状態保持に利用します。

※ 補足 (Phase 216 監査による追記): 上記のうち「ニックネーム」「服薬履歴データ」「家族メンバー情報 (表示名)」は、Apple の App Privacy 区分上 NSPrivacyCollectedDataTypeOtherUserContent (ユーザー生成コンテンツ) としてまとめて宣言しています。本アプリは HealthKit を利用しないため、服薬履歴は「健康データ (Health and Fitness)」ではなく「ユーザーコンテンツ (Other User Content)」として取り扱います。

(3) 当社が取得しない情報

  • 広告識別子 (IDFA) — 本アプリは広告 SDK を導入していないため取得しません (NSPrivacyTracking = false)。
  • 位置情報 — 本アプリは位置情報の取得 API を使用しません。
  • HealthKit データ — 本アプリは HealthKit を統合しないため、ヘルスケアアプリ経由のデータを一切取得しません。
  • 連絡先 / 写真ライブラリ / カメラ / マイク — 本アプリは利用しません。
  • 第三者分析・トラッキング SDK (Google Analytics, Firebase, Sentry, Crashlytics 等) — 一切導入していません。

第3条 (個人情報の利用目的)

当社は、取得した個人情報を以下の目的でのみ利用します。

  1. アカウント管理および本人認証 — メールアドレス、パスワード、ユーザーID。
  2. 本サービスの機能提供 — 服薬履歴の記録および家族メンバー間での共有 (履歴データ、家族メンバー情報)。
  3. コミュニティ機能の提供 — レビューの投稿および閲覧 (ニックネーム、投稿コンテンツ)。
  4. 本サービスの品質向上および障害解析 — MetricKit による匿名化されたパフォーマンス計測。
  5. 法令上の義務の履行 — 警察その他の公的機関からの法令に基づく開示請求への対応。

第4条 (第三者提供)

当社は、ユーザーの同意なく個人情報を第三者に提供しません。ただし、次のいずれかに該当する場合はこの限りではありません。

  1. 法令に基づく開示請求があった場合。
  2. 人の生命、身体または財産の保護のために必要があり、ユーザーの同意を得ることが困難な場合。
  3. 本サービスの提供に必要な範囲内で、第5条に定める委託先に個人情報を取り扱わせる場合。

第5条 (個人情報の取扱いの委託 / 第三者サービス利用)

当社は、本サービスの提供にあたり、以下の第三者サービスを利用します。各サービスにおける個人情報の取り扱いは、各サービスのプライバシーポリシーに従います。

サービス 提供事業者 用途 プライバシーポリシー
Supabase Supabase Inc. (米国) 認証 / データ同期 / ファイルストレージ https://supabase.com/privacy
App Store / StoreKit Apple Inc. (米国) アプリ配信 / サブスクリプション課金 https://www.apple.com/legal/privacy/
Vercel Vercel Inc. (米国) ウェブ版のホスティング / 配信 https://vercel.com/legal/privacy-policy

ユーザーがコミュニティ機能を利用してレビュー等を投稿した場合、その内容は Supabase が管理するデータベースに保存され、当社サービスを利用する他のユーザーから閲覧可能となります。

第5条の2 (安全管理措置)

当社は、取得した個人情報の漏えい、滅失またはき損の防止その他の安全管理のために、以下の措置を講じます。

  • 通信の暗号化 — 本サービスと当社が利用するクラウド基盤との通信は、すべて TLS (HTTPS) により暗号化されます。
  • 保管データの保護 — パスワードはハッシュ化して保管し、原文を保持しません。iOS アプリでは認証情報を端末の安全な領域 (キーチェーン) に保管します。
  • アクセス制御 — データベースは行レベルセキュリティ (Row Level Security) により、各ユーザーが原則として自己のデータのみにアクセスできるよう制限しています。
  • 最小限の取得 — 本サービスの提供に必要な範囲を超える個人情報は取得しません。
  • 委託先の監督 — 第5条に定める委託先に対し、適切な安全管理が図られるよう必要な監督を行います。

第6条 (Cookie およびトラッキング技術)

本サービスは、広告目的または行動追跡目的の Cookie・トラッキング技術を一切使用しません。iOS アプリは、Apple の App Tracking Transparency フレームワーク (ATT) で定義されるトラッキングを行いません (NSPrivacyTracking = false)。

ウェブ版では、ユーザーのログイン状態を保持するために必要な機能的な Cookie (認証セッション Cookie) のみを使用します。これらは本人認証の目的にのみ用いられ、広告配信や第三者による行動追跡には利用されません。

第7条 (健康情報・要配慮個人情報の取り扱い)

本サービスは、ユーザーが入力する服薬履歴を「ユーザーコンテンツ」として取り扱います。Apple HealthKit との連携は行わず、本アプリのデータが Apple ヘルスケアアプリへ送信されることはありません。

ユーザーが本サービスに入力する服薬履歴、病名・症状に関する記載その他の健康に関する情報は、個人情報の保護に関する法律 (以下「個人情報保護法」といいます) 第2条第3項に定める「要配慮個人情報」に該当する場合があります。当社は、これらの要配慮個人情報を、ユーザーが本サービスの登録時および利用時に同意したうえで、本サービスの機能提供の目的に限り取得・利用します。

これらの情報の入力は任意であり、ユーザーは入力しないことを選択できます。また、入力した情報はいつでも本サービス上で閲覧・訂正・削除することができます。当社は、これらのデータを医療目的で利用または分析することはなく、ユーザーの同意なく第三者に提供しません。

第8条 (個人情報の保存期間)

  • アカウント情報および当社が管理するすべての関連データは、ユーザーがアカウント削除を実行した時点から30日以内に Supabase 環境から削除されます。
  • 本アプリのアンインストールにより、端末上 (SwiftData) に保管されたキャッシュデータは消去されます。
  • 法令により保存が義務付けられている情報は、法定の期間に従い保管します。

第9条 (ユーザーの権利)

ユーザーは、当社が保有する自己の個人情報について、以下の権利を有します。

  1. 開示請求 — 当社が保有する個人情報の開示を求める権利。
  2. 訂正・追加・削除請求 — 個人情報が事実と異なる場合の訂正等を求める権利。
  3. 利用停止・消去請求 — 個人情報の利用停止または消去を求める権利。
  4. 同意の撤回 — 本ポリシーへの同意を将来に向けて撤回する権利 (撤回後は本サービスの利用ができなくなります)。

これらの権利は、本アプリの「マイページ → アカウント削除」または tmedilab.support@gmail.com への連絡により行使できます。当社は、本人確認のうえ、遅滞なく対応します。

第10条 (未成年者のプライバシー)

本アプリは App Store における年齢区分「4+」で提供されますが、特に未成年者を対象とした設計ではありません。13歳未満の方が本アプリを利用される場合は、保護者の同意を得たうえでご利用ください。当社は、保護者の同意なく13歳未満の方の個人情報を意図的に収集しません。

第11条 (国外への移転)

本サービスのデータは、主として Supabase のアジア太平洋 (東京) リージョンのデータセンターに保管されます。ただし、本サービスの提供にあたり、第5条に定める米国に所在する事業者 (Supabase Inc.、Vercel Inc.、Apple Inc. 等) がデータの処理・保管に関与する場合があります。

ユーザーは、本サービスの利用にあたり、個人情報が国外に所在する事業者により取り扱われる場合があることに同意するものとします。当社は、これらの委託先が個人情報保護法およびその関連ガイドラインに適合する適切な保護措置を講じるよう努めます。

第12条 (本ポリシーの改定)

当社は、必要に応じて本ポリシーを改定することがあります。重要な変更がある場合は、本アプリ内通知または当社ウェブサイト上で30日以上前に告知します。改定後の本ポリシーは、当社が指定する効力発生日から適用されます。
