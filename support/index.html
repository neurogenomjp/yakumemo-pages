処方メモ — サポート

処方メモをご利用いただきありがとうございます。よくあるご質問への回答およびお問い合わせ窓口をご案内します。

お問い合わせ

  • メールアドレス: tmedilab.support@gmail.com
  • 本アプリ内の動線: 「設定 → 規約・ポリシー → お問い合わせ」をタップすると、ブラウザで本ページが開きます (Phase 212 / 2026-06-16 で配線)。

返信は通常 2 〜 5 営業日以内にお送りします。土日祝日のお問い合わせは翌営業日以降の対応となります。

よくあるご質問

Q1. 課金したのに Premium 機能が解放されない

以下をお試しください。

  1. 本アプリ右上のメニューから「マイページ → Premium を復元」をタップしてください。
  2. iOS の「設定 → Apple ID → サブスクリプション」で、Yakumemo Premium が有効になっているかご確認ください。
  3. 上記でも解決しない場合、Apple ID のメールアドレスおよび購入日時を添えて tmedilab.support@gmail.com までご連絡ください。

Q2. サブスクリプションを解約したい

サブスクリプションの管理および解約は、iOS の「設定 → Apple ID → サブスクリプション」または以下のリンクから行えます。

  • https://apps.apple.com/account/subscriptions

当社では Apple ID 経由のサブスクリプションを直接停止することはできませんので、ご了承ください。

Q3. 家族メンバーを追加できない / 上限が表示される

無料プランでは家族メンバーの登録上限が設定されています。Premium プランにご加入いただくと上限が拡大されます。

最大登録数は 5 名です。

Q4. PDF 出力ができない

PDF 出力は Premium プランの機能です。無料プランではご利用いただけません。

Q5. お薬情報が古い・誤っているように見える

本アプリのお薬マスタは、独立行政法人 医薬品医療機器総合機構 (PMDA) が公開する医薬品情報を元に編集しています。情報の誤り、最新情報の反映漏れにお気づきの場合は、対象のお薬名と内容を添えて tmedilab.support@gmail.com までご連絡ください。

なお、本アプリの情報は医療従事者による診断・処方の代替ではありません。実際の服薬に関する判断は、必ず医師または薬剤師にご相談ください。

Q6. 不適切なレビューを通報したい

該当のレビュー画面で「通報」ボタンをタップしてください。当社で内容を確認のうえ、利用規約に違反すると判断した場合は速やかに対処します。

特定のユーザーの投稿を非表示にしたい場合は、「ブロック」機能をご利用ください。

Q7. アカウントを削除したい

本アプリ内の「マイページ → アカウント削除」からお手続きください。アカウント削除後、当社が管理するすべての関連データは 30 日以内に削除されます。

Q8. パスワードを忘れた

ログイン画面下部の「パスワードを忘れた方」リンクから、登録メールアドレス宛にパスワードリセット用のメールをお送りします。

動作環境

  • iOS 18.0 以降
  • iPhone (推奨)

iPad では iPhone レイアウトで動作しますが、iPad 向けに最適化された UI は v1 ではご提供しておりません。

既知の制限 (v1.0)

  • v1 では APNs によるプッシュ通知に対応していません (アプリ内通知のみ)。
  • iPad 専用レイアウトには未対応です。
  • Sign in with Apple および Google ログインには未対応です (メールアドレス/パスワード認証のみ)。

これらは順次対応を予定しています。
